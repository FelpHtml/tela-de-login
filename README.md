# 🔐 Tela de Login

Este projeto consiste em uma tela de login simples desenvolvida utilizando apenas **HTML5** e **CSS3**.

Foi criado com o objetivo de praticar conceitos básicos de estruturação de páginas, estilização e organização do código durante meus estudos de desenvolvimento web.

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3

## 📷 Preview

## ✨ Funcionalidades

- Interface de login responsiva
- Campos para e-mail e senha
- Botão de login
- Layout moderno e limpo

## 🎯 Objetivo

Este projeto faz parte dos meus estudos em desenvolvimento front-end, buscando aprimorar conhecimentos em:

- Estrutura semântica com HTML
- Estilização com CSS
- Organização de arquivos
- Boas práticas de desenvolvimento
